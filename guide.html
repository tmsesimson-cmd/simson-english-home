“수험영어를 넘어 세상을 보는 창” — 입학 후 꼭 확인해야 할 홈페이지·앱 사용법과 학원 생활 안내입니다.

학습 숙지사항 관리 숙지사항 캠퍼스 안내

PART 1

학습 숙지사항

홈페이지·모바일 앱에서 학습을 시작하기 위한 필수 단계입니다.

01

정보수정

  • 등록 시 받은 임시 아이디/비밀번호를 기억하기 편한 것으로 변경합니다.
  • 홈페이지와 모바일 앱 모두 동일하게 변경할 수 있습니다.
⚠ 정보수정은 전자도서관 레벨테스트 이전에만! 반 배정 후 변경하면 전자도서관 이용 불가 + 이전 기록이 삭제됩니다.
02

학생기록카드 작성 · 동영상 시청

  • 로그인 → 학생기록카드 작성 후 맨 아래 동의칸 체크
  • 첨부된 광고 동의서는 확인·서명 후 학생을 통해 제출
  • [나의 동영상]에서 안내 영상 5개 필수 시청
03

과제확인과 통합자료실

  • 이름 밑 [과제확인] 버튼 → 달력에서 날짜 선택
  • 숙제 기한 / 숙제 파일 / 온라인 과제 내용 확인
  • 필요한 음원(MP3)·뉴스 동영상은 [통합자료실]에서 과목별 다운로드
  • 숙제 완료 후 오른쪽 [Self Check] 반드시 완료
04

데일리 테스트 · 재시 결과 확인

  • 데일리 테스트: [과제확인] 버튼 아래에 쪽지시험 결과 표시
  • 재시: [재시/출석] → 재시 결과·평가·상세 확인
05

포트폴리오 확인

  • 디베이트: 해당 달 촬영 영상(같은 반 학생 모두 포함)
  • 뉴스: 해당 달 Ani&News 개인 촬영 영상
  • 에세이: Debate 시간 원어민 선생님 첨삭본
06

전자도서관 — 레벨테스트 & 책 읽기

  • 학생 아이디로 이용 / [전자도서관] → [Level Test] → Grade·Month 선택
  • 학년별 레벨: 1~3학년 = K · 4~5학년 = 1 · 6학년 = 2
  • 책 읽기: [Study Room] → 읽고 [Test] 80점 이상 합격(기회 2번)
  • [My Score]에서 읽은 책 확인, 뉴스 공책에 기록
⚠ 레벨테스트 기회는 1번입니다. 시험 도중 Skip 버튼을 누르지 마세요.
07

선생님 상담

  • [선생님 상담] → [글쓰기]로 담임·과목 선생님께 학습 질문 및 답변

PART 2

관리 숙지사항

출결·셔틀·수강등록·휴퇴원 등 학원 생활에 필요한 안내입니다.

01

출결 관리

  • 지각·결석 시 수업 시작 10분 전까지 사유를 미리 작성
  • 앱: [결석미리알리기] / 홈페이지: 달력의 보라색 [출결알림] 버튼
02

셔틀 이용 안내

  • 홈페이지 [학원소개]→[셔틀안내] / 앱 [메뉴]→[셔틀확인]
  • [캠퍼스 선택]→[호차 선택] 후 정류장·시간·기사/보조선생님 연락처 확인
  • 예정 시간 3분 전 대기, 차량 주목(핸드폰·실내 대기 자제)
  • 재시로 남을 경우 다음 셔틀 또는 개별 하원, 지정 정류장에서만 승·하차
03

수강등록 · 결제 안내

  • 홈페이지 [MY심슨]→[수강료&교재] / 앱 [수강료결제]
  • 선납 기준, 매월 20~27일 등록 (지연 시 5일부터 홈페이지 사용 제한)
  • 교재비는 새 학기 개강 시 발생, 과목·반별 추가 구입 가능
  • 카드결제 또는 가상계좌 (현금영수증 번호는 가입 전화번호로 자동 등록)
04

휴 · 퇴원 규정

  • 휴원: 증빙 가능한 사유로 증빙 첨부 시 가능
  • 초1~초5 연 2회, 초6 연 1회 / 최대 한 달 / 3·6·9·12월 불가
  • 휴원 중 테스트·홈페이지 사용 불가, 다음 학기 반 배정 불이익 가능
  • 수강료 선납·익월 이월 가능(퇴원 시 교육청 환불 규정 적용)
  • 퇴원: 2개월 이후 재입학 가능
05

주지사항

  • ‘산업안전보건법’에 따라 모든 통화 내용이 녹음됩니다.
  • 긴급 상황 시 [선생님 상담]란에 ‘긴급상황’으로 메모를 남겨 주세요.
  • 별첨(재원생 규정·장학 규정·셔틀 운행 규정·개인정보 동의서)에 서명 후 제출
